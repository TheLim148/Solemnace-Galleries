# Solemnace Galleries

Unofficial fan catalog site about Trazyn's Infinite Galleries from Warhammer 40,000. In the future it's planned to place canonical and original exhibits from the collection here with descriptions, categories and sources.


## Project status

Development is temporarily **frozen**. Project may be continued later.


## Stack

- HTML
- CSS
- JavaScript - planned.
